# wowoRoom 床墊電商平台 (純 JavaScript 開發 )

- [前台Demo](https://youen1126.github.io/WOWOROOM-2026-haxSchool/)
- [後台Demo](https://youen1126.github.io/WOWOROOM-2026-haxSchool/admin.html)

- 純 JavaScript 開發，無使用任何框架。
- 包含完整前後台頁面模擬與訂單資料、圖表呈現。
- 本作品採用現成模板實作功能練習，實作以JS邏輯為主，不包含版型切版。

## 前台特色

- axios 串接商品 API 並渲染畫面
- 購物車加入與數量調整、刪除購物車
- 表單處理
- 避免重複請求 API

## 後台特色

- axios 串接訂單 API 並渲染畫面
- 訂單狀態切換
- 使用 C3.js 呈現營收統計圖表：產品類別營收比重、熱門商品營收比重

## 使用套件

- axios
- SweetAlert2
- C3.js 圖表
